Git credentials: avoid inserting username and password every time

1min read

November 10, 2019

Are you annoyed because git keeps asking for credentials every time after a git push or pull?

There are a couple of solutions:

  1. SSH keys
  2. git credentials

Usually I always used the first method, but recently I found this answer on Stack Overflow and it seems that with this single command git will enable the helper that we need:

git config --global credential.helper store

Now issue git pull o git push and inert your credentials.
After this git will save them and you will never see another details promtp again!

I’m using this method since a month and never had an issue, hope this will be helpful also to you!
