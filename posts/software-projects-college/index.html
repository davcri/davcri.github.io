Four software projects I developed during college

2min read
[software]

January 20, 2018

During my time in Computer Engineering college in Italy, I developed various kind of software projects as an assignment for the courses.

Softhub - Bachelor’s Thesis

https://github.com/davcri/softhub

Softhub is a web application developed as a part of my Bachelor’s Thesis. The goal was to design and implement a maintainable and scalable application.

I picked up Python and the Django web framework. They both have a great documentation that helped me to develop the webapp withouth reinventing the wheel for common tasks; this saved me a lot of time that I spent focusing on:

  • Creating cohesive and decoupled Python classes using Object Oriented Analysis and Design (OOA and OOD) and GRASPs;
  • Automating the installation and configuration, using Python virtualenv, Django custom admin commands and Bower. Probono appreciated this effort:

For who doesn’t know Probono, he is the developer of the great Appimage packaging format for Linux applications!

Unibox - Web development

https://github.com/davcri/Unibox-Ajax

Unibox was my first web development project. You can found it here (http://uniboxaq.altervista.org/), but the user interface is avaliable only in Italian. The application is based on the MVC architecture, a quite commond Design Pattern for web applications.

I realized a single page web application using only JavaScript, jQuery and AJAX requests: it was a fun learning experience.

Pong UDP - Computer Networking

https://github.com/davcri/pong-clone

Pong UDP is a simple game with a client/server multiplayer architecture. For the (very basic) graphics we used libGDX Java game framework.

A simple game network protocol was designed to manually take care of packets validation and ordering at application level. We used UDP and JSON to implement it.

Pushup app - Personal summer project

https://github.com/davcri/Pushup-app

Pushup app let’s you keep track of progress in a pushup program.

I developed it to experiment with desktop applications and graphic toolkits.

It uses Qt4 (now obsolete), matplotlib, and sqlite.

Conclusions

During college it was quite common to study physics, electronics, mathematics and other subjects while studying programming languages and tinkering with software.

Even if these applications were developed with tight time constraints, I’m pretty happy for the results, considering also that it was often a solo effort.
